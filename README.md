
# Getting Started

## Step 1: Install npm pachage 

```bash
npm install
```

## Step 2: Install pods

```bash
cd ios
pod install
cd ..
```

## Step 3:  Start the Metro Server

```bash
npm start
```

## Step 4:   Start your Application

### For Android

```bash
npm run android
```
### For iOS

```bash
npm run ios
```
